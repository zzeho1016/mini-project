# 네이버 뉴스 헤드라인 크롤러

네이버 뉴스 메인 페이지에서 헤드라인 기사를 수집하여 터미널에 출력하고, 날짜별 '.txt' 파일로 저장

## 사용 기술

- Python 3.x
- requests
- BeautifulSoup4
- datetime

## 실행 방법

```bash
pip install -r requirements.txt
python news_crawler.py

```
